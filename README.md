# web

Source for [idan.goldman.work](https://idan.goldman.work) — my personal thoughts, notes, and projects.

This is a community mirror of my private repository, here so curious folks can read behind the scenes, ask questions, or write suggestions.

## Stack

Built with [Hugo](https://gohugo.io) (extended) and the
[Stack](https://github.com/CaiJimmy/hugo-theme-stack) theme by Jimmy Cai, then
built and deployed by [Forgejo Actions](https://forgejo.org) to [Cloudflare Pages](https://pages.cloudflare.com/).
The whys and hows live in my [notebook](https://idan.goldman.work/notebook/).
